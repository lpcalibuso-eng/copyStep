1. Stage the specific file:
git add "[file path]"

example:
git add "/opt/lampp/htdocs/Prototype System/Step/kldstep/resources/js/Pages/Adviser/Permission.jsx"


Noted: Gawin mo para madali right click mo yung file or kaya yung folder na gusto mong i-push tas click mo yung "copy path" tas paste mo sa loob ng double quote
  
2. Commit the File:
git commit -m "Message mo(kahit ano pero sabihin mo ano update dito)" 

3. Verify the Remote Connection:
git remote -v

4. Push to the Remote Repository:
git push origin <your-branch-name>

branch naten:
    main
    my new feature

    Recommend ko ito para mas madali ang push:
    git push origin HEAD
